# flet-2048
Гра 2048 на Python Flet

### Алгоритм ходу вліво

**1. Стиснення** -- зсуваємо всі ненульові числа вліво:
```
[0, 2, 0, 4]  ->  [2, 4, 0, 0]
```

**2. Злиття** -- зліва направо зливаємо сусідні рівні числа:
```
[2, 2, 4, 4]  ->  [4, 0, 8, 0]
```

**3. Повторне стиснення:**
```
[4, 0, 8, 0]  ->  [4, 8, 0, 0]
```
